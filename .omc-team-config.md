**目标:** Footer.tsx、Stats.tsx 改为深色玻璃态，与现有组件风格一致

**已确认风格（参考现有组件）:**
- Navbar.tsx: background: rgba(0,8,20,0.8), backdrop-blur-xl, border neon-green/20
- Features.tsx: background #030a14, card: border border-neon-green/30 bg-slate-900/50
- Hero.tsx: 深色背景 + ParticleCanvas (100粒子+鼠标交互+neon绿连线)
- GallerySection.tsx: 深色背景 #030a14
- CtaSection.tsx: 深色背景 + neon 按钮

**Footer.tsx 修改要求:**
- background: rgba(0,8,20,0.9)
- border-top: 1px solid rgba(0,255,163,0.2)
- 文字颜色: text-neon-green/80 或 text-neon-green/60
- 链接 hover: text-neon-green
- Logo 区域保持 neon-green 强调色

**Stats.tsx 修改要求:**
- background: 深色玻璃态
- 数字: text-4xl font-bold text-neon-green
- 标签: text-sm text-neon-green/60
- 卡片布局: 与 Features 卡片风格一致（border + bg-slate-900/50）

**完成后:** build 验证 `cd ~/Downloads/spro && npm run build`

### Executor-2: GSAP 动画集成
**目标:** 将 lib/animations.ts 中定义的动画应用到各组件

**lib/animations.ts 已有内容（确认后使用）:**
```typescript
// 查找实际内容
cat ~/Downloads/spro/lib/animations.ts
```

**应用位置（优先顺序）:**
1. Hero.tsx - Title/Subtitle 渐入动画（gsap.from 配合 stagger）
2. Features.tsx - 卡片依次入场（ScrollTrigger）
3. GallerySection.tsx - 横向滚动增强
4. CtaSection.tsx - 按钮脉冲动画
5. Stats.tsx - 数字计数器动画

**注意:**
- 使用 'use client' 标记动画组件
- ScrollTrigger 需要注册: gsap.registerPlugin(ScrollTrigger)
- 动画应该柔和（neon 风格不宜过快）
